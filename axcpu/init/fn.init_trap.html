pub fn init_trap()
Expand description

Initializes trap handling on the current CPU.

In detail, it initializes the GDT, IDT on x86_64 platforms (init_gdt and init_idt). If the uspace feature is enabled, it also initializes relevant model-specific registers to configure the handler for syscall instruction (init_syscall).
